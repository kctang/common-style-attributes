ng-style-attributes

HTML attributes to manage layout and common styling properties for HTML elements.

By using these attributes, we should be able to handle 80% of an application styling needs. For the rest, you can always use CSS/SCSS directly.

Why? Because styling every HTML component with CSS/SCSS just too tedious, hard to maintain and easily leads to styling inconsistencies.

Getting Started
  • This library was designed to be used with Angular applications.
