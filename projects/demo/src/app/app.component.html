common-style-attributes

HTML attributes to manage layout and common styling properties for HTML elements.

Why? Because styling every HTML component with CSS/SCSS is tedious and hard to maintain. It leads to styling inconsistencies as project grows. Let's avoid that. Keep custom CSS/SCSS to the 20% of advanced styling needs. For the other 80%, use common style attributes.

Getting Started
  • Install from NPM:

    npm install common-style-attributes

  • Add this to a SCSS file in your project:

    @import '~common-style-attributes/styles/common-style-attributes';

  • To use typography styles, add this to a SCSS file in your project:

    @import '~common-style-attributes/styles/csa-typography';

  • SCSS variables can be used to customize typography styles. To do this, create a new SCSS file based on ~common-style-attributes/styles/csa-variables.scss and import it to your project:

    @import 'my-csa-variables';
